## 용어 정리

### 릴리즈 노트 구성 항목

- 릴리즈 노트 : 고객에게 개발 과정에서 정리된 제품의 릴리즈 정보를 제공하는 문서
- 헤더 : 릴리즈 노트 이름 등 기존 정보가 포함

### 애플리케이션 테스트의 기본 원칙

- 결함 집중 : 에러의 80%는 20%의 소스코드에 있다 (pareto법칙)
- 살충제 패러독스 : 반복적인 테스트 실행은 똑같은 에러를 제외한 다른 에러는 발생하지 않는다.

### 데이터 마이닝

> 마이닝(mining) : 광물을 캐다

- 대량의 데이터 안에서 체계적인 패턴이나 규칙을 찾아네는 기술

### 프로토콜

- 톰마릴, 컴퓨터가 메시지를 전달하고 메시지가 제대로 도착했는지 확인, 도착하지 않았을 경우 메시지를 재전송 하는 인련의 방법 '기술적 은어'

- 3가지 구성 요소
  - 구문(Syntax)
  - 의미(Semantics)
  - 타이밍(Timing)

### 해시(Hash) 암호화 방식

- SHA 시리즈, MD5, N-NASH, SNEFRU
- MD5 : RFC 1321로 지정, 주로 프로그램이나 파일이 원본 그대로인지를 확인하는 무결성 검사, 예전에 쓰이던 MD4를 대체

### 랜드 어택(Land Attack)

- 공격자가 패킷의 출발지 주소나 포트를 임의로 변경하여 출발지와 목적지 주소를 동일하게 함으로써, 공격 대상 컴퓨터의 실행 속도를 느리게 하거나 동작을 마비시켜 서비스 거부 상태에 빠지도록 하는 공격

### 반정규화

- 시스템의 성능 향상, 개발 및 운영의 편의성 등을 위해 정규화된 데이터 모델을 통합, 중복, 분리 하는 과정으로, 의도적으로 정규화 원칙을 위배

### DRS (재해 복구 시스템, Disaster Recovery System)

- RTO(Recovery Time Objective) : 복구시간 목표
  - 비상사태 또는 업무중단 시점으로부터 업무가 복구되어 다시 정상가동 될 때까지의 시간
- RPO(Recovery Point Objective) : 복구 시점 목표
  - 업무 중단 시 각 업무에 필요한 데이터를 여러 백업 수단을 이요하여 복구할 수 있는 기준점

### 네트워크 계층 프로토콜

- IP : 송수신간의 패킷단위
- ARP : 네트워크상에서 IP주소를 MAC주소로 변환
- RARP : MAC주소는 알지만 IP 주소를 모르는경우
- ICMP(Internet Control Message Protocol) : 문제를 알려주는 메시지 형식 프로토콜
- IGMP(Internet Group Management Protocol) : 인터넷 그룹 관리
